## 前言

这本书的排版与文字矫正还在不断地完善中，力求与原著一模一样。这本书中的内容针对的是 水浒传 的 施耐庵 版本。

《水浒传》以农民起义的发生、发展过程为主线，通过各个英雄被逼上梁山的不同经历，描写出他们由个体觉醒到走上小规模联合反抗，到发展为盛大的农民起义队伍的全过程，表现了“官逼民反”这一封建时代农民起义的必然规律，塑造了农民起义领袖的群体形象，深刻反映出北宋末年的政治状况和社会矛盾。
